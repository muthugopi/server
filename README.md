**Project Overview**

This project is a practice backend server built with Node.js and MySQL.
It supports:

CRUD operations for users and students

Query filtering, pagination, and sorting

Basic authentication middleware

MySQL database integration

Input validation using express-validator

It is meant for learning and testing backend development concepts.

Features

Users API: Add, update, patch, delete, and list users

Products API: List products and get product by ID

Students API: Fetch, add, update, and delete student records in MySQL

Protected routes: Simple token-based authentication

Query features: Filtering, sorting, and pagination

Logging middleware: Tracks request duration

Error handling: Centralized error responses

Tech Stack

Node.js

Express.js

MySQL (mysql2)

express-validator for validation

dotenv for environment variables

morgan for logging

cors for Cross-Origin Resource Sharing
